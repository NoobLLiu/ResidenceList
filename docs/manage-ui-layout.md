- **分页区域**：当前为 slot 27-44（2行×9列=18格，扣除翻页按钮后实际16格放评论）
- **翻页按钮**：slot 36(上一页) / slot 44(下一页)
- **空位填充**：所有未设置的 slot 用 `PluginConfig.ICON.EMPTY` 填充
- **权限管理子菜单**（3行箱子）：slot 0=返回, slot 11=全局权限, slot 15=玩家权限

### 设计原则

1. **领地图标单独置于第一行正中间（slot 4）**，突出展示领地核心信息
2. **顶部两行（Row 0-1）** 放置所有功能按钮，按类别从左到右分组
3. **中间两行（Row 2-3, slot 18-35）** 作为评论分页区域，每页可显示 18 条评论
4. **翻页按钮紧贴评论区下方（Row 4）**，左端=上一页，右端=下一页
5. **底部一行（Row 5）** 全部用 `EMPTY` 填充，作为视觉边框
6. 类别之间用空位填充（`EMPTY`）作为视觉分隔
